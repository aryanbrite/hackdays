## About the page

### No Scroll

The user dont need to scroll at the page anywhere. They can just click on the buttons and get the information

### Animation

I have used javascript to remove and add class for animation to each and every element you can see on the website. 
Position, Animation css were used extensively while making this site

### Contents

The page has six layer of contents. 

- Prizes
- Where ?
- When?
- Timeline 
- Venue (locked)
- Sponsors


### Small things which took time

This took me around 14 hours to make this, I haved tried to give in,out and hover or atleast one of them to give the kind of animation I wanted this site to have

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- Vercel

I made this using only custom css. No tailwind/frameworks and components
